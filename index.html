Teste A/B Para um Jogo Mobile.

O objetivo desse projeto era projetar um Teste A/B para um Jogo Mobile. À medida que os jogadores avançam no jogo, eles encontrarão portões que os obrigam a esperar algum tempo antes de poderem avançar ou fazer uma compra no aplicativo. Neste projeto, analisaremos o resultado de um teste A/B em que o primeiro portão em Cookie Cats foi movido do nível 30 para o nível 40. Em particular, analisaremos o impacto na retenção dos jogadores.

Foram usadas ferramentas como Python, bibliotecas de Visualização de Dados e conceitos de Estatística para entender, modelar e aplicar o Teste A/B.

O resultado concluiu que houve um efeito considerável ao mover o portão para o nível 40. Porém esse efeito traria uma diminuição na receita, sendo mais sensato manter o portão como estava no nível 30.

As ferramentas utilizadas foram:

  • Python, Pandas, Matplotlib, Seaborn
  • Jupyter Notebook
  • Git,Github
  • Estatística(Teste Chi-Squared)

Ranqueamento de Clientes para uma empresa de Seguros de Saúde.

Utilizei também, algoritmos de Machine Learning, mais especificamente os Supervisionados de Classificação.

O resultado foi uma planilha no Google Sheets que, dado uma base de clientes, fazia a previsão através de uma API que buscava o modelo de Machine Learning na Nuvem e retornava essa base clientes ordenada por maior probabilidade de adquirir o seguro de carro.

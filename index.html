Projetos em Ciência de Dados

Ranqueamento de Clientes para uma empresa de Seguros de Saúde.

O objetivo desse projeto foi ajudar uma empresa de seguros de saúde a vender também seguros de carro para os clientes de sua base, ou seja, fazer um Cross-Sell. Para isso foi feito um modelo de Machine Learning que ranqueia os clientes que tem a maior probabilidade de adquirir o seguro de carro, facilitando a estratégia de comunicação da empresa, otimizando seu negócio e reduzindo custos.

Foram usadas ferramentas como Programação em Python para Análise Exploratória e Manipulação de Dados, Conceitos de Estatística para construir Insights de negócio, ferramentas web como Github e servidores na Nuvem como o Heroku. Utilizei também, algoritmos de Machine Learning, mais especificamente os Supervisionados de Classificação.

O resultado foi uma planilha no Google Sheets que, dado uma base de clientes, fazia a previsão através de uma API que buscava o modelo de Machine Learning na Nuvem e retornava essa base clientes ordenada por maior probabilidade de adquirir o seguro de carro.

As ferramentas utilizadas foram:

  • Python
  • Jupyter Notebook
  • Heroku Cloud
  • Machine Learning
  • Github
  • Estatística

Previsão de Vendas de uma Rede de Farmácias.

Nesse projeto foi feita a previsão de vendas da rede de farmácias da Rossmann nas próximas 6 semanas que pode ser acessada por um bot no Telegram de qualquer dispositivo com acesso à internet.

Foram usadas ferramentas como Programação em Python para Análise Exploratória e Manipulação de Dados, ferramentas web como Github e ORender. Foram também utilizados conceitos e modelos de Machine Learning como XGBoost, Random Forest, entre outros.

O resultado final foi um Bot no Telegram que acessa o modelo de Machine Learning publicado na Nuvem e da informações sobre a previsão de vendas de determinada loja.

As ferramentas utilizadas foram:

  • Python
  • Jupyter Notebook
  • Render Cloud
